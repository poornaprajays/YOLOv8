YOLO Family Timeline

YOLOv1 2016 Original — grid-based, single pass
YOLOv3 2018 Multi-scale anchors, Darknet-53
YOLOv5 2020 PyTorch, popular production choice
YOLOv8 ⭐ 2023 Anchor-free, C2f backbone, best accuracy/speed

YOLOv8 Model Variants

Choose the right model for your use case. Larger = more accurate but slower. yolov8n is what this project uses — it's great for learning!

Model Size Parameters mAP50-95 Speed (CPU) Best For
yolov8n ← this project 6 MB3.2M37.3~1.4ms Learning, real-time on CPU

yolov8x 136 MB68.2M53.9~13.4ms Maximum accuracy, GPU required
mAP50-95 = Mean Average Precision at IoU thresholds from 0.50 to 0.95. Higher is better. Measured on COCO val2017.
